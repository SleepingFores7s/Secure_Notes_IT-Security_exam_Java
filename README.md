
# Secure Notes - Java Console Application

This is my Secure Notes project, built as part of my Java course.
The goal was to create a simple console application where users can register, log in, and manage their own notes, all stored in a MySQL database.

The app runs entirely in the console and uses menus for navigation.
Passwords are hashed, users have roles, and everything is handled through stored procedures in the mySQL database.


## Installation

    1. Clone the repo
    2. Import into IDE
    3. Create the mySQL database
    4. Update the Database credentials in DatabaseConnection.java
    5. Run Main.java
